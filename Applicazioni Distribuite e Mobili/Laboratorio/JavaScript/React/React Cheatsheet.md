# React Cheatsheet

## Generics

### Create Project
```bash
$> npx create-react-app $PROJECT_NAME [--use-npm]
```

### Start React Project
```bash
$> npm start
```

## Install Packages
```bash
$> npm install $PACKAGE_NAME
```

### Useful Packages

#### Material UI (Despite React Version)
```bash
$> npm install @material-ui/core --legacy-peer-deps
```

#### React Router DOM
```bash
$> npm install react-router-dom
```
